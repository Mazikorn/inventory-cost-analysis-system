| TC-001-02 | 创建抽盘任务 | 1. 点击"创建盘点任务"<br>2. 输入任务名称<br>3. 选择"抽盘"模式<br>4. 选择物料类型<br>5. 点击创建 | 任务创建成功，仅包含选定类型物料 | 符合预期 | ✅ 通过 |
