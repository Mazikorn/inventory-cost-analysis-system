## 2. 开发环境配置

### 2.1 硬件要求

| 组件 | 最低配置 | 推荐配置 |
|------|---------|---------|
| CPU | 4核 | 8核+ |
| 内存 | 8GB | 16GB+ |
| 硬盘 | 50GB SSD | 100GB SSD |
| 网络 | 10Mbps | 100Mbps |

### 2.2 软件要求

| 软件 | 版本 | 用途 |
|------|------|------|
| Node.js | v18.17.0+ | 运行时环境 |
| npm | v9.6.0+ | 包管理器 |
| PostgreSQL | v15.0+ | 数据库 |
| Redis | v7.0+ | 缓存 |
| Git | v2.40.0+ | 版本控制 |
| Docker | v24.0+ | 容器化（可选） |

### 2.3 前端配置

**文件**: `frontend-v2/.env.development`

```env
NODE_ENV=development
REACT_APP_API_BASE_URL=http://localhost:8080/api
REACT_APP_VERSION=v1.1.0-dev
REACT_APP_ENV=development

REACT_APP_ENABLE_DEBUG=true
REACT_APP_ENABLE_MOCK=true
REACT_APP_MOCK_DELAY=500

REACT_APP_SENTRY_DSN=
REACT_APP_GA_TRACKING_ID=
```

**安装依赖**:

```bash
cd frontend-v2
npm install
```

**启动开发服务器**:

```bash
npm run dev
```

### 2.4 后端配置

**文件**: `backend/.env.development`

```env
NODE_ENV=development
PORT=8080

DATABASE_URL=postgresql://postgres:password@localhost:5432/coreone_dev
REDIS_URL=redis://localhost:6379/0

JWT_SECRET=dev-jwt-secret-change-in-production
JWT_EXPIRES_IN=7d

LOG_LEVEL=debug
LOG_FORMAT=combined

CORS_ORIGIN=http://localhost:3000

UPLOAD_DIR=./uploads
MAX_FILE_SIZE=10485760

ENABLE_SWAGGER=true
ENABLE_GRAPHQL=true
```

**安装依赖**:

```bash
cd backend
npm install
```

**数据库迁移**:

```bash
npm run db:migrate
npm run db:seed
```

**启动开发服务器**:

```bash
npm run dev
```

### 2.5 数据库配置

**创建数据库**:

```sql
CREATE DATABASE coreone_dev;
CREATE USER coreone_user WITH PASSWORD 'password';
GRANT ALL PRIVILEGES ON DATABASE coreone_dev TO coreone_user;
```

**初始化脚本**:

```bash
psql -U postgres -d coreone_dev -f scripts/init-db.sql
```

### 4.4 数据脱敏

**脱敏规则**:

| 数据类型 | 脱敏方式 | 示例 |
|---------|---------|------|
| 患者姓名 | 替换为随机姓名 | 张三 → 李四 |
| 患者ID | 替换为随机ID | P001 → P999 |
| 手机号 | 中间4位替换为* | 13812345678 → 138****5678 |
| 身份证号 | 中间10位替换为* | 110101199001011234 → 110101********1234 |
| 价格 | 随机偏移 ±10% | 580.00 → 638.00 |

**脱敏脚本**:

```bash
npm run db:anonymize:staging
```

### 5.4 安全配置

**HTTPS 配置**:
- 使用 Let's Encrypt SSL 证书
- 自动续期
- 强制 HTTPS 重定向

**防火墙配置**:
- 仅开放 80、443 端口
- 数据库端口仅内网访问
- SSH 端口限制 IP 白名单

**备份策略**:
- 数据库每日全量备份
- 文件每日增量备份
- 备份保留 30 天

## 8. 常见问题

### 8.1 开发环境问题

**问题**: 数据库连接失败  
**解决**: 检查 PostgreSQL 服务是否启动，检查连接字符串是否正确

**问题**: Redis 连接失败  
**解决**: 检查 Redis 服务是否启动，检查端口是否正确

**问题**: 前端无法访问后端 API  
**解决**: 检查 CORS 配置，检查后端服务是否启动

### 8.2 测试环境问题

**问题**: 测试数据污染  
**解决**: 运行数据清理脚本，重新导入测试数据

**问题**: 测试环境性能慢  
**解决**: 检查服务器资源，优化数据库查询
